To view a demo, run the code within file demo.m

The demo requires 3 additional images.

1. ref_mask_1 (background mask) : a copy of ref with mask pixels (with value 255)
				  marking the background.
2. ref_mask_2 (nucleus mask) : a copy of ref with mask pixels (with value 255)
				  marking the nucleii
3. ref_mask_3 (unstained mask) : a copy of ref with mask pixels (with value 255)
				  marking the unstained cells
